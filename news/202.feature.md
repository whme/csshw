The currently selected client in the `[e]nable/disable input`
control-mode submenu is now highlighted with a configurable
console color (`client.highlighted_console_color` in
`csshw-config.toml`), making it easier to track which window
`[e]`, `[d]`, and `[t]` will act on. Pressing one of those keys
briefly flashes the resulting state color on the highlighted
window before restoring the highlight, giving the user visual
confirmation that the action landed.
