`cargo xtask generate-social-preview` now renders the GitHub social preview card from an HTML template via headless Chromium in a Playwright Docker container, replacing the previous pre-rendered PNG stamping approach. The card pulls live star count, description, and language distribution from the GitHub API. The host only needs Rust, Cargo, and Docker — no local Node.js or Playwright install required. New flags: `--out <PATH>` and `--token <TOKEN>`.
